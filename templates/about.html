About This Project

This web application allows users to explore and manage Seattle-based Airbnb listings and reviews. It was built using Flask and integrates two types of databases: a MySQL database hosted on AWS RDS for storing listings and reviews, and AWS DynamoDB for user account information.

The platform is designed for smooth navigation and user interaction. Once logged in, users can view listings based on filters like room type and neighborhood, write and manage reviews, and update or delete their account.

Main Features:

  • Filter listings by room type and neighborhood
  • Submit, view, and delete your own reviews
  • Manage account information (change password, delete account)
  • Data is pulled dynamically from MySQL and DynamoDB
